# EventTracker
Proyecto en grupo para el segundo parcial de la materia: **Diseño de Interfaces**. Solo con HTML y CSS.

El sitio web se trata de un buscador de eventos, de todo tipo de categorías. Dónde se puede obtener información de la fecha, lugar y artistas.
El sitio web tiene las siguientes características:

- Home.
- About us.
- Tres páginas de eventos.
- Un buscador, con filtros.
- Login.
- Formulario de contacto.
